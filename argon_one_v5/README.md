# Argon ONE V5 Add-on

Controls and monitors the Argon ONE V5 case OLED display.

## Screen Rotation

The OLED display rotates through three screens:

1. **Temperature & Fan Status**
   - Current CPU temperature
   - Fan status indicator
   - Current time

2. **System Resources**
   - CPU usage percentage
   - RAM usage (percentage and MB)

3. **Home Assistant Status**
   - Add-on active status
   - System uptime
   - Current date

## Logs

View the add-on logs to see:
- OLED initialization status
- Screen rotation events
- Periodic system status (every minute)
- Any errors encountered
